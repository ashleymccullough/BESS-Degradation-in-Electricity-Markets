This repository hosts the code and data relevant to recreate degradation capacity loss and operating profiles for a generalized BESS system in power markets.

The "McCullough_BESS_OptimalSchedululingWithDegradation.ipynb" Jupyter Notebook is the main file used to generate these results.

All CSV files hold yearly LMP power pricing information for all US ISOs.

All XLSX files hold experimental testing data and summarized information used by the main Jupyter Notebook for the specific case study outlined.
